# Kompilacja i uruchomienie

## wymagane

-   node 14
-   npm 7
-   weryfikacja/update `.env` (albo https://stackoverflow.com/questions/43069087/pass-shell-environment-variable-as-argument-in-npm-script)

## dev

1. `npm install`
2. `npm start`

## prod

1. `npm ci`
2. `npm run build`
3. czekamy na katalog(i) `dist(s)`
