# Nu installation example

## Requirements:

* Docker (Engine `24.x.x` or newer)

## Running:

```bash
./start.sh
```

## Accessing:

- Nu Designer: http://localhost:8080
- Flink console: http://localhost:8080/flink
- AKHQ console: http://localhost:8080/akhq

## Cleaning up:

```bash
./stop-and-clean.sh
```
